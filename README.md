## Getting Started

Clone this repository

```bash
git clone git@github.com:maplessmann/weather-app.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run start
```

Open the local server at `localhost:8000`

- [React](https://reactjs.org/)
- [Gatsby](https://www.gatsbyjs.com/)
